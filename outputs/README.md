# Outputs

This folder contains selected non-sensitive example outputs from strategy research and backtesting.

Current sample files:

- `nvda_equity_sample.csv` — example equity curve from an NVDA threshold strategy backtest
- `nvda_trades_sample.csv` — example completed trade log from the same backtest
- `nvda_equity_voltp_sample.csv` — example equity curve from a volatility/take-profit variant
- `nvda_trades_voltp_sample.csv` — example trade-leg output from the volatility/take-profit variant

These outputs are included for portfolio demonstration only. They should not be interpreted as live trading performance, investment advice, or proof of future profitability.
